# Audivex `v1.4.2`

Audivex est une application de bureau qui vous permet de télécharger et convertir vos vidéos YouTube en MP3 ou MP4, simplement et rapidement.

## Fonctionnalités

- Collez un lien YouTube, choisissez MP3 ou MP4, téléchargez
- MP3 haute qualité avec pochettes et paroles
- MP4 avec plusieurs qualités (360p à 1080p)
- Playlists complètes en un clic
- Sous-titres disponibles pour les MP4
- Mises à jour automatiques
